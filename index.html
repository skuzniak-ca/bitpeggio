▶ WHAT DOES IT ALL DO?
WAVEFORM
The raw tone color. PULSE = punchy square (NES lead/melody), TRI = soft triangle (NES bass), SAW = bright and buzzy, NOISE = pitched static (drums/explosions).
PULSE WIDTH
Only affects PULSE. Sets the duty cycle - how much of each cycle is "on." 50% is a full square; lower values get thin and nasal. Sweep it while holding a note for that classic chiptune morph.
ENVELOPE (ADSR)
Shapes a note's volume over time. Attack = fade-in time, Decay = drop after the peak, Sustain = held level while a key is down, Release = fade-out after you let go.
MASTER
VOL = overall loudness. OCT = which octave the keys play (lower = deeper). Change it with the arrows or the Z / X keys.
PATCHES
One-click sounds that set the waveform + pulse width + envelope for you. A fast way to hear what the controls can make.
SCOPE
An oscilloscope - a live picture of the sound wave you're playing. Watch how each waveform and the envelope change its shape.

▶ WHAT DOES IT ALL DO?
TONE
The base waveform each voice is built from. Two of them are layered and slightly detuned for a fuller, chorus-y 16-bit body. SAW = strings/brass, TRI = soft/flutey, SQR = reedy.
DETUNE
How far apart the two stacked oscillators are (in cents). More detune = a wider chorus. Zero = clean and thin.
WARMTH (LOW-PASS)
A low-pass filter that rolls off the highs. Lower = darker and warmer, closer to the SNES's muffled character; higher = brighter.
REVERB
Wet/dry mix of a reverb tail - the spacious "big room" sound the SNES was known for. Turn it up for pads and choirs.
VIBRATO
A gentle pitch wobble (an LFO) added to every note for expression.
ENVELOPE (ADSR)
Shapes a note's volume over time: Attack, Decay, Sustain level, Release tail. Slow attacks make sweeping pads.
MASTER / PATCHES / SCOPE
VOL and OCT as in 8-bit. Patches are one-click sounds. Scope shows the live (post-reverb) waveform.
